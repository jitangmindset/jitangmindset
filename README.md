## What I care about

- **Product completeness.** Installers, migrations, backups, documentation and failure states matter as much as the main screen.
- **Explicit trust boundaries.** Renderers do not get filesystem access, browser data does not reach a vault directly, and diagnostic tools do not execute supplied code.
- **Useful interfaces.** Clear hierarchy, keyboard paths, restrained motion and readable dark/light themes.
- **Verification.** Small deterministic tests for parsers and security boundaries, plus integration and visual smoke checks where unit tests are not enough.
